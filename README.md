
# Telegram SMM Bot

Этот бот предназначен для автоматической генерации постов с текстом и изображениями для Telegram на основе заданной темы.

## Функционал

*   **Генерация текста поста**: Использует модель Mistral AI (`mistral-small-latest`) для создания живого, цепляющего текста на заданную тему (1500-2000 знаков, с эмодзи и Markdown-форматированием).
*   **Генерация изображения**: Использует агент Mistral AI (`mistral-medium-2505`) для создания релевантного изображения к посту.
*   **Отправка в Telegram**: Публикует сгенерированный пост (изображение с подписью и продолжение текста, если он длинный) в указанный чат Telegram.

## Настройка и запуск

Для работы бота вам потребуется:

1.  **Telegram Bot Token**: Создайте нового бота через BotFather в Telegram и получите его токен.
2.  **Mistral AI API Key**: Получите API-ключ от Mistral AI для доступа к их моделям.

### Локальный запуск (или в среде, отличной от Colab)

1.  **Установите зависимости**:

    ```bash
    pip install pyTelegramBotAPI mistralai html2text markdown
    ```

2.  **Сохраните код бота**: Создайте файл `bot.py` и скопируйте в него содержимое соответствующего кода.

3.  **Укажите токены**: Вставьте ваши `TELEGRAM_BOT_TOKEN` и `MISTRAL_API_KEY` в соответствующие переменные в файле `bot.py`.

    ```python
    TELEGRAM_BOT_TOKEN = "ВАШ_ТЕЛЕГРАМ_ТОКЕН"
    MISTRAL_API_KEY = "ВАШ_MISTRAL_API_КЛЮЧ"
    ```

    *Примечание*: Для большей безопасности рекомендуется хранить токены в переменных окружения или использовать систему секретов.

4.  **Запустите бота**:

    ```bash
    python bot.py
    ```

Бот запустится и начнет опрос Telegram API на наличие новых сообщений. Отправьте ему команду `/start` или нажмите кнопку 'Создать пост', чтобы начать взаимодействие.

## Использование

1.  Отправьте боту команду `/start`.
2.  Нажмите кнопку 'Создать пост'.
3.  Введите тему или запрос для генерации поста (например, 'Преимущества удаленной работы' или 'Космические путешествия').
4.  Бот сгенерирует текст и изображение, а затем опубликует их в чате.
